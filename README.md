# 小说阅读室

*******************


## 运行环境
iOS 9.0+

## 功能简介
设备安装App进入指定的四本小说即可一起阅读，同步当前阅读进度和背景字体设置，还可以一起文本交流（同步和消息是基于环信MQTT实现），原本是打算做成匹配（两人）的方式一起阅读小说，但是时间有限，就先做成了多人一起阅读（阅读室）

体验场景：设备A点击小说《末法时代》，设备B也点击进入《末法时代》，设备A和设备B会彼此会同步阅读进度等操作，也可以发消息交流。
